TypeScript 與 SQL Server 的型態對應

發表於 2024-02-12

我知道 TypeScript 的型態很少，但就稍微做個紀錄

TypeScript 類型 SQL Server 類型
number Int, Decimal, Float
string Char, VarChar, Text
boolean Bit
Date Date, Datetime

簡單的先列一下，原則上都是以上表格所列的這些類型或基於以上類型的拓展(NChar, NVarChar, SmallDateTime, BigInt…等)，原則上大致猜的出來。真的沒辦法就用 any 幹過去吧 🤣
